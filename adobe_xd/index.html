Adobe XD is a vector-based user experience design tool for web apps and mobile apps.

Website Layout

When starting a new design best to use the dimensions of W: 1400 & H: 800.

Grid

This gives you some margins to keep everything within the middle of the page. Use the parameters below

Grid: Layout
Colums: 2
Gutter Width: 1140
Column Width: 130
Margins Linked: 0
