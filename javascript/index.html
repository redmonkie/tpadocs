    String Length/Number Retrival

    Below is code I made to ask a message, display how many characters are in the message and work out the remaining characters if 280 is max.

    // Prompt that feeds into variable.
var message = prompt("Whats your beef?");
// Variable that works out remainder
var remainder = 280 - message.length
// Display message
alert("You have written " + message.length + " characters, you have " + remainder + " left.");

    It can also be written simpler by doing the maths within the alert using ().

    var message = prompt("Whats your beef?");

alert("You have written " + message.length + " characters, you have " + (280 - message.length) + " left.");

    String Slicing

    Using the slice() method you can specify what part of a string you want to include. Starts at 0.

    var name = "Marselle"
name.slice(2,3)
> "r"

// Range (Start at 3 up to but not including 7)
name.slice(3,7)
> "sell"

    Below will slice the output to be maximum of 140 characters

    var message = prompt("Whats your beef?");

var message = message.slice(0,140)

alert(message);

    Single line version of above which will:
    1. Prompt
    2. Slice
    3. Alert 

    alert(prompt("Whats your beef?").slice(0,140));

    Change String Case

var name = "Marselle"

> "marselle"

    String Case and Slice

    By using slice we can isolate which part of the variable we make upper case and whats lower case.
    Also as we don't know the length of the users name by starting the slice at 1 and then using the .length method we say make the second character and then the length of the variable lower case.

    var name = prompt("What is your name?");
alert(name.slice(0,1).toUpperCase() + name.slice(1,name.length).toLowerCase());

    Maths

    Look at Python Maths for all mathmatical syntaxs.

    There's an order of presendence. *, % before +, -.
    5 * 2 would come before 3 + 5

    var cost = 3 + 5 * 2

    If you wanted 3 + 5 to come first you would encase it in brackets.

    var cost = (3+5) * 2

    It's good practice to add the parenthesis even though you know the multiplication or divison is happening first.

    var cost = 3 + (5 * 2)

    Increment/Decrement

    To increase a value use += & *=. To decrease a value use -= & /=.

y -= 6;
