    Lists & Dictionaries

    Lists

    They use square brackets.

    userAge = [18, 19, 22, 25, 30]

    You can print a particular value in a list.

    print(userAge[1])
19
print(userAge[0])
18
print(userAge[-1])
30

    You can have empty lists and fill them later with the apend() function

    emptyList = []

    Make a list into a variable (variable = list)

    userAgeV1 = userAge

    Assign index to a variable. Notation 2:4 is known as a slice.

    userAgeV2 = userAge[2:4]

    print (userAgeV2)
[22, 25]

    The third number in the slice is a stepper. We get a sublist consisting every second number.

    userAgeV3 = userAge[1:5:2]
print (userAgeV3)
[19, 25]

    Appending to a list

    router =['891','2901', '2951', '4001']

    Append to existing list

    router.append('2950')
#Append a list to a list
router.append(['mx240', 'srx200'])

    Slice notations have useful defaults. The default for the first number is zero, and the default for the second number is size of the list being sliced. For instance, userAge[ :4] gives you values from index 0 to index 4-1 while userAge[1: ] gives you values from index 1 to index 5-1 (since the size of userAge is 5, i.e. userAge has 5 items).

    Tuple

    Tuples are just like lists, but you cannot modify their values. The initial values are the values that will stay for the rest of the program.

    To declare a tuple, you write tupleName = (initial values). Use parentheses ( ) when declaring a tuple.

    Multiple values are separated by a comma.

    monthsOfYear = ("Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul", "Aug", "Sep", "Oct", "Nov", "Dec")
print(monthsOfYear[4])

    Dictionaries

    Dictionary is a collection of related data PAIRS.

    userAge = {'Bjorn':32, 'Hatti':18, 'Matthew':44}

    Or via a method

    userAge = dict(Bjorn = 32, Hatti = 18, Matthew = 44)

    To print a users age

    print(userAge['Bjorn'])

    Create empty dictinoaries to be filled in later

    userName = dict()

    Add to empty dictionary

    userName['John Smith'] = 'jsmith'

    Delete from dictionary

    del userName['John Smith']

    Display list in easy to read format

    userCricket = {
    'Brian Lara' : 'blara',
    'Courtney Walsh' : 'cwalsh',
    'Curtley Ambrose' : 'cambrose',
    'Malcom Marshall' : 'mmarshall',
}
